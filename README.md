# Modern Recipe Book

A modern, responsive recipe web application built with **React**, **Vite**, and **Tailwind CSS**.  
This project demonstrates frontend architecture, state management, and polished UI/UX design using a real-world content system powered by structured JSON data.

 **Live Demo:** https://kent-cookbook.netlify.app/

---

## Highlights

- Dynamic recipe pages with ingredient scaling
- Interactive ingredient checklist with state management
- Ingredient info popups with reusable data architecture
- Search by title and tags
- Mobile-first responsive design
- Static deployment with optimized performance (Netlify)

---

## Tech Stack

- React (Vite)
- Tailwind CSS v4
- React Router
- Static JSON data layer
- Netlify deployment

---

## What This Project Demonstrates

- Component-driven frontend architecture
- Clean separation of data and presentation
- Efficient client-side filtering and UI state handling
- Modern responsive design principles
- Production-ready static deployment workflow
